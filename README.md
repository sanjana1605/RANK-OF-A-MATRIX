## Step 5:

## Step 4:

## QUESTION:
```
Write a program to find the rank for the given matrix([5,-3,-10],[2,2,-3],[-3,-1,5]).
```
## Program:
```

Developed by:Sanjana Sri N
RegisterNumber:2305003007  

```
`````python
import numpy as np
A=np.array([[5,-3,-10],[2,2,-3],[-3,-1,5]])
rank=np.linalg.matrix_rank(A)
print(rank)
`````
